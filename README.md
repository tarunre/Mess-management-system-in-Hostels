# Messo – Mess Management System

#  Overview

Messo is a full-stack Mess Management System developed to simplify and automate mess operations in educational institutions and hostels. The system allows users to view menus, submit complaints, receive announcements, and provide reviews, while administrators can efficiently manage mess-related activities.

# Features

- User Registration and Login using JWT Authentication
- Daily Menu Management
- Complaint Registration and Tracking
- Announcement System
- HR Review Management
- Secure Authentication and Authorization
- Responsive User Interface

# Technologies Used
